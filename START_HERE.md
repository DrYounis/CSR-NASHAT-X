## 🎬 فيديو سريع (خطوات مصورة)

### الخطوة 1: فك الضغط
```
📁 ramadan_sports_bot.zip
  └─> فك الضغط
      └─> 📂 ramadan_sports_bot/
```

### الخطوة 2: افتح Terminal
- **Windows:** اضغط Win+R → اكتب `cmd` → Enter
- **Mac:** اضغط Cmd+Space → اكتب `Terminal`
- **Linux:** Ctrl+Alt+T

### الخطوة 3: انتقل للمجلد
```bash
cd ramadan_sports_bot
```

### الخطوة 4: ثبّت المكتبات
```bash
pip install -r requirements.txt
```

### الخطوة 5: أنشئ ملف .env
```bash
cp .env.example .env
```

ثم افتح `.env` بأي محرر نصوص وضع مفاتيحك

### الخطوة 6: شغّل البوت
```bash
python ramadan_bot.py
```

اختر رقم 1 للتجربة، أو رقم 4 للنشر التلقائي
